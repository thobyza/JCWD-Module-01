switch case:
- kalau banyak kondisi disarankan pakai switch, instead of else if
- switch case -> kondisi nya harus sama dengan (tidak bisa lebih besar/ lebih kecil (>=, <=))

while & do While:
- while    : mikir dulu baru ngerjain (ngecek kondisi dulu, baru aksi)
- do while : kerja dulu baru mikir    (aksi dulu, baru ngecek kondisi) 

For Loop:
- syntax dan cara berpikir alurnya:
    for (bikin variable; ngecek kondisi; maka... (aksinya) apa seteah looping sebelumnya) {
        aksi jika kondisi = true
    }

    for (bikin variable (i); selama i masih... ; maka... ) {
        aksi jika kondisi i = true
    }

- continue = untuk ngeskip/continue ke loop selanjutnya tanpa menjalankan aksi
